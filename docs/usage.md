# 使用说明

本文档面向实际打麻将时的手机端使用场景。

## 访问应用

部署用户建议从 [GitHub Releases](https://github.com/littleseven2003/mahjong_scoreboard/releases) 下载稳定版本源码包后启动服务。

本地开发环境访问：

```text
http://localhost:8899
```

Docker 部署后访问：

```text
http://服务器局域网IP:8899
```

同一张麻将桌上的玩家需要连接同一个局域网，并访问同一个地址。

## 合规声明

本软件工具仅用于线下娱乐麻将场景中的积分记录与结算辅助，与任何赌博、资金交易或其他违法行为无关。使用者应遵守所在地法律法规，不得将本项目用于任何违法违规用途。

## 创建房间

1. 打开首页。
2. 点击“创建房间”进入创建页。
3. 选择三人或四人模式。
4. 填写房间名称和房主昵称。
5. 设置起始分、计分单位、是否允许负分。
6. 点击“创建房间”。

创建成功后，页面会进入房间等待页，并显示 6 位房间号。

房主可以点击“查看二维码”。其他玩家可以使用系统相机或浏览器扫码打开加入链接，也可以在“加入房间”页面点击“扫码填写”调用摄像头识别二维码。二维码链接会进入加入房间页面，并自动填入房间号。

## 继续对局

首页会显示当前浏览器加入过、且尚未结束的对局。

未完成对局从创建或开始后超过 12 小时，会在继续对局列表中标记为“已超时”。超时项会显示“移除”按钮，用于删除本机保存的继续入口。

移除只清理当前浏览器的入口记录，不会删除服务器上的房间或历史数据。

## 加入房间

1. 打开首页。
2. 点击“加入房间”进入加入页。
3. 输入房间号，或点击“扫码填写”扫描房主展示的二维码。
4. 输入玩家昵称；如果留空，会使用输入框中的默认昵称。
5. 点击“加入房间”。

如果使用系统相机扫描房主二维码，会直接打开加入房间页面并自动填入房间号。浏览器内扫码需要浏览器允许摄像头权限。部分手机浏览器只允许 HTTPS 或 `localhost` 页面调用摄像头；如果局域网 HTTP 访问无法打开摄像头，建议使用系统相机扫码。

昵称在同一个房间内不能重复。项目当前没有账号系统，玩家身份保存在当前浏览器的本地存储中。

玩家在等待阶段退出后，该昵称会被释放，其他设备可以使用同一个昵称重新加入。

## 准备与退出

等待阶段中，房主不需要准备。

非房主进入房间后可以：

- 点击“准备”；
- 点击“取消准备”；
- 点击“退出房间”。

当房间人数已满，且除房主外的所有玩家都准备完毕后，房主会收到提示。

房主可以在等待阶段随时解散房间。房间解散后，房间内所有玩家都会收到“房间已解散”的提示，并回到首页。

## 开始对局

房间处于等待状态，人数已满，且所有非房主玩家都已准备时，房主可以点击“开始对局”。

后端要求房间人数满足创建时选择的三人或四人模式，并且所有非房主玩家都已准备，才能开始对局。

## 记分

对局开始后，页面顶部会显示所有玩家当前分数。

记录一笔分数：

1. 点击“积分变动”面板中的“记录我的扣分”。
2. 在弹窗中以按钮形式选择加分玩家。
3. 输入分数，分数框只接受数值，默认留空并在确认时校验。
4. 可选填写备注，例如“点炮”“杠分”“其他”。
5. 点击“确认变动”。

非房主只能以自己作为扣分玩家。房主可以点击“记录我的扣分”或“为他人扣分”。房主在“为他人扣分”弹窗中以按钮形式选择扣分玩家和加分玩家，扣分玩家与加分玩家互斥，已选扣分玩家在加分玩家栏中不可点击。

提交后，后端会：

- 写入流水；
- 扣减扣分玩家分数；
- 增加加分玩家分数；
- 向房间内所有客户端广播最新状态。

确认变动或取消弹窗后，分数输入会恢复为空。

## 撤销流水

流水列表中的每一笔有效流水都可以申请撤销。只有该笔流水的扣分玩家和加分玩家都确认后，系统才会真正撤销并回滚分数。

撤销不会删除原始流水，而是把流水标记为已撤销，并记录双方确认状态。这样可以保留对账痕迹，也避免单方误撤销。

当一方发起撤销后，另一方会看到撤销确认弹窗，可以选择：

- 确认撤销：同意撤销；
- 取消撤销：拒绝本次撤销申请；
- 暂时关闭：先关闭弹窗，流水栏中仍保留处理按钮。

如果另一方取消撤销，本次撤销申请会失效，需要重新申请。

## 结束结算

房主点击“结束结算”后，对局状态变为已结束，并生成每位玩家的结算结果：

- 起始分；
- 最终分；
- 输赢分；
- 按计分单位换算后的结果。

所有玩家的输赢分总和应为 0。

## 历史记录

首页右上角可以进入“历史”。

历史记录展示已结束对局，点击某一局可以查看：

- 玩家结算；
- 完整流水；
- 已撤销流水标记。

## 使用限制

- 不计算麻将番型。
- 不判断胡牌、自摸、点炮等麻将规则。
- 没有用户登录和跨设备身份恢复。
- 默认用于局域网娱乐积分记录，不建议直接暴露到公网。
